Frequently Asked Questions

What file types does Mediary support?

Mediary supports EPUB and PDF for books, and MP4 and MOV for videos.

How do I import files into Mediary?

You can import files via iTunes File Sharing, the iOS Files app, AirDrop, or by tapping "Open in Mediary" from any supported app.

How many files can I import for free?

The free plan allows up to 5 files. Upgrade to Premium for unlimited imports with a one-time purchase.

How do I restore my purchase?

Open Mediary, go to the Premium screen, and tap "Restore Purchases". Make sure you are signed in with the same Apple ID used for the original purchase.

Does Mediary require an internet connection?

No. Mediary is completely offline. Your files are stored on your device and do not require an internet connection to access.

How do I use Picture in Picture?

While playing a video, press the home button or swipe up to go to the home screen. The video will automatically continue playing in a floating window.

How do I delete a file?

In your library, swipe left on any file and tap Delete, or tap the trash icon at the top of the All Files screen.

My book or video won't import. What should I do?

Make sure the file format is supported. If the issue persists, try re-downloading the file and importing again. Contact us below if the problem continues.
